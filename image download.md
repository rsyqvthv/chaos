
# batch image download extensions for chrome
ImageAssistant Batch Image ownloader ](https://chrome.google.com/webstore/detail/imageassistant-batch-imag/dbjbempljhcmhlfpfacalomonjpalpko)

* 筛选, 取图无可挑剔
* 尺寸略大
* 但是批量下载, 对于不希望设置不提示下载的用户来说就很痛苦.

[Image Downloader Continued](https://chrome.google.com/webstore/detail/image-downloader-continue/jfkjbfhcfaoldhgbnkekkoheganchiea)

* 滑块筛选不错
* 筛选很好
* 53k, 尺寸小巧
* 功能略比上个小, 但足够用
* 批量没有存页面, 需要设置不提示下载.

[Imageye - Image downloader
](https://chrome.google.com/webstore/detail/imageye-image-downloader/agionbommeaifngbhincahgmoflcikhm)

* 好评如潮的
* 没排序
* 界面不太适合显示多图

[image-picker
](https://chrome.google.com/webstore/detail/image-picker/bhibldekjicdbnjeeecmgoogcihoalhe)

* 这个功能阳春
* 306k 
* 只能单选下载

[Download All Images
](https://chrome.google.com/webstore/detail/download-all-images/ifipmflagepipjokmbdecpmjbibjnakm)

* 没有筛选, 一股脑下载.
* 长处是自动打包zip下载
* 85.79KiB

[Fastezt - Save Images Photos on Websites
](https://chrome.google.com/webstore/detail/fastezt-save-images-photo/bfmhfjabfdlpjjhffgegfopapdldkdgp)

* 只有大图/全部图两个选择
* 下载也需要设置非提示下载

[Image Downloader
](https://chrome.google.com/webstore/detail/image-downloader/dheemnnkahegifficiphgacgjfmigjec)

* 过滤需要手动输入, 点击按钮
* 没有根据url过滤
* 没有排序
* 但是下载直接zip打包
* 不改文件名

[Image Downloader
](https://chrome.google.com/webstore/detail/image-downloader/pfkpgccmeafapmchhknifkncoehlnnfj)

* 操作流畅
* 可跨tab操作
* 有高级搜索模式
* 滑块过滤, 排序都有
* 没有url过滤
* 下载也需要设置非提示.
* 303KiB
