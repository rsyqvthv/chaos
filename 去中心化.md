由 Mozilla 联合创始人 Brendan Eich 所创公司开发的 Brave [成为首个支持 IPFS 点对点协议的浏览器](https://brave.com/ipfs-support/)。IPFS 代表 InterPlanetary File System，类似 BitTorrent 的点对点协议，设计作为去*中心化*储存系统，允许用户在数百甚至数千个节点之间共享不受审查的内容。当 Brave [探测](https://blog.ipfs.io/2021-01-19-ipfs-in-brave/)到用户试图访问 IPFS 内容或使用原生地址如 ipfs:// 或 ipns://，浏览器会提示用户安装和启用 IPFS 节点，或使用一个 HTTP 网关。
