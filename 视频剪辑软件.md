# 视频剪辑软件

- pr的难度太大

- 微软自家的moviemaker

- Shotcut [官网](https://www.shotcut.org/) 开源、跨平台、自带中文、功能齐全(*免费*中的专业级)
  然而并不易用（对新手而言，这是个人体验）

- 用ffmpeg

- 我是感觉vegas真的不错

- [视频编辑王](https://www.apowersoft.cn/video-editor): 注册个账号，就能直接*免费*使用了

- 爱剪辑

- Windows Live [软件包](https://support.microsoft.com/zh-cn/help/18614/windows-essentials)中的影音制作，基本剪辑都没问题，还有自动音量平衡。加字幕样式也还算足够。唯一的问题就是偶尔会崩溃，所以要记得随时保存。
  汉化的，中文版自称为“Windows 影音制作”

- SMMVSplitter: ru的神器, 专门对付不转码的视频切割.

- kdenlive（windows+linux）

- 蜜蜂剪辑（windows，apowersoft出品，可以很简单地破解，直接买有学生优惠）

- bandicut 都是不转码剪辑的

- openshot (2.4.4)

- shotcut (19.10.20)

- 剪映，手机版本和mac版本已经出了。
