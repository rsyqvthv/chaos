# 问题
* 当代ai最根本的问题， 莫过于个深度学习在数据的变化面前十分脆弱， 很容易被加入的噪声糊弄。
* 当下的AI可以被称为狭义的AI。  他们的根本问题是从数据里挖掘关联性， 而不是因果。
* 深度学习终归属于统计学习

# 解决
> 将当下的深度学习整合经典符号主义AI的思想=下一代AI

深度学习的著名批评者Gary Marcus指出未来AI发展方向。将当下的深度学习整合经典符号主义AI的思想， 让整个系统变得更加稳健， 具有真正智能体的泛化能力。

那么如何赋予AI这种能力？他认为分为四个步骤：

1. 规定知识的基本形式
2. 能够把基本知识整合成无穷多知识（涵盖整个世界）的操作系统
3. 通过知识组成成模型 
4. 利用知识和模型进行认知和推理
