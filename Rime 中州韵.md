# Rime 输入法

rime的版本就够理解一阵子的, 这是我见过的最差的版本管理. 搜索的时候可以用'小狼毫'和'rime'

1. Linux：
   - 官方：`ibus-rime`（中州韵）
   - 第三方：`fcitx-rime`
2. windowns：
   - 官方：`Weasel`（小狼毫）
   - 第三方：`PRIME`
3. macOS：
   - 官方：`Squirrel`（鼠须管）
   - 第三方：`XIME`
4. Android：
   - 第三方：`Trime`（同文）
5. iOS：
   - 第三方：`iRime`
